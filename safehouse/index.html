SafeHouse: An IoT-Based Biometric Security System

SafeHouse is a complete home security platform integrating IoT sensors, solenoid lock control, and biometric access with an ML-enabled dashboard.

  • PIR motion sensor event monitoring
  • Secure biometric login with MTCNN + FaceNet + k-NN
  • Flask backend dashboard for real-time alerts and control
  • 99.4% recognition accuracy with fast retraining pipeline
